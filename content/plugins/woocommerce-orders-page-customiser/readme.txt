Allows users to manage WooCommerce Orders page to reorder, resize and enable/disable columns.

== Description ==

WooCommerce Order Page Customiser allows users to manage the columns on the Orders page. You can choose which columns to enable/disable, as well as resize and reorder them.

This extension is WooCommerce 2.0.x compatible.

== Usage ==

Simply upload the plugin and you're good to go. There are no options to configure - the extension just works.
